# 国家政策证据库

本文件夹用于支持 `report.md` 中的行业趋势判断、城市定位判断和职业路径排序。

## 使用原则

- 优先使用中国政府网、国家发展改革委、工业和信息化部、自然资源部、科技部、国家数据局等官方来源。
- 政策原文优先于解读稿，规划纲要优先于新闻报道。
- 每个政策结论都需要记录来源、发布日期、发布机关、适用周期和证据等级。
- 政策只能说明国家资源配置方向，不能直接等同于就业机会；后续需与招聘数据、公司岗位和在职者访谈交叉验证。

## 证据等级

| 等级 | 含义 | 使用方式 |
|---|---|---|
| A | 中央/国务院/部委正式原文，或多个官方文件一致 | 可直接进入行业/城市判断 |
| B | 官方解读、地方正式规划、部委专题说明 | 可作为强辅助证据 |
| C | 官方新闻稿、统计公报、会议通稿 | 只作为趋势补充 |
| D | 媒体报道、券商研报、个人推测 | 不进入政策证据库核心表 |

## 输出目标

- `policy_catalog.csv`：原始政策文件索引。
- `05_提炼结果/行业趋势总表.md`：从政策中提取行业趋势。
- `05_提炼结果/城市定位总表.md`：从政策中提取城市/区域定位。
- `05_提炼结果/城市_产业_岗位矩阵.md`：把城市定位、产业特点和候选岗位连接起来。
